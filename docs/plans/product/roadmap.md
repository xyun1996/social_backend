# Product Roadmap

## Goal

Build a real, maintainable social backend product from the current prototype assets, starting with a reduced runtime topology and a smaller but deeper feature surface.

## Milestones

1. Runtime consolidation
2. Product foundation rebuild
3. Phase A core social package
4. Staging and release readiness

## Acceptance

- The new runtime topology exists and is the active target in documentation.
- Shared foundation work moves to the new runtime line first.
- The first product implementation line focuses on login, friends, blocks, invites, private chat, guild basics, and party basics.

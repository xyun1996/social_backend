# V1 Tasks

Store one executable task per file in this directory.

## Naming

- Recommended format: `NNN-short-task-name.md`
- Example: `001-bootstrap-gateway-module.md`

## Required Fields

- title
- status
- version
- milestone
- background
- goal
- scope
- non-goals
- dependencies
- acceptance criteria
- related docs / ADRs

## Status Values

- `planned`
- `in-progress`
- `done`
- `blocked`
